# Mania de Mulher — MVP (protótipo de aprovação)

Mockup estático de loja (Home, Coleção e Produto) com a identidade visual da Mania de Mulher.
Arquivo único e autossuficiente: `index.html` (HTML + CSS + JS inline).
Imagens vêm do CDN da Tray e as fontes do Google Fonts, então funciona em qualquer host.

## Como publicar (GitHub Pages)
1. Suba o `index.html` na RAIZ de um repositório público.
2. Settings -> Pages -> Source: Deploy from a branch -> branch `main` / pasta `/(root)` -> Save.
3. A URL https://SEU-USUARIO.github.io/NOME-DO-REPO/ fica no ar em ~1 min.

## Alternativas
- Netlify Drop: arraste o index.html
- tiiny.host: upload do index.html
